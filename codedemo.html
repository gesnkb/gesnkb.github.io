Wondering about the NavBar at the top of the screen?

A navigation bar is a great tool for any web developer to use in their websites

Navbars are how you can keep a theme between pages, and allow accessible navigation between different webpages. Styling a navbar however, can be tricky. It requires some special CSS attributes that make it function properly. One of those is the position: sticky The CSS for my navbar looks like this:

#navbar{ background-color: #ccffff; width: 100%; height: auto; color: #000000; position: sticky; }

Click on all of the links within the navbar to navigate between pages!
